# Natural Language Processing in Mental Health

## Integrating Clinical and Non-Clinical Text for Depression Detection

---

## Project Overview

This project investigates the integration of clinical and non-clinical text for depression detection using Natural Language Processing (NLP).

The study compares three integration strategies:

- Sequential Fine-Tuning
- Multi-Task Learning
- Hybrid Learning

The project evaluates whether integrating multiple text domains improves depression classification performance compared with single-domain learning.

---

## Research Objectives

- Review existing NLP methods used in mental health.
- Develop an integrated framework for clinical and non-clinical text.
- Implement multiple learning strategies.
- Evaluate model performance.
